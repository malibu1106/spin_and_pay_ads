Politique de Confidentialité

Dernière mise à jour : 01/03/2026

Bienvenue dans Spin & Pay. Votre vie privée est importante pour nous. Cette politique explique comment les données sont traitées, notamment dans le cadre de la diffusion publicitaire.

1. Collecte des données

Nous ne collectons pas directement de données nominatives (nom, email, etc.). Cependant, pour assurer le fonctionnement de l'application et la diffusion de publicités, des données techniques sont collectées automatiquement par nos partenaires :

  • Identifiants publicitaires : Identifiant publicitaire Google (AAID) ou Apple (IDFA).
  • Données techniques : Modèle de l'appareil, version du système d'exploitation, et adresse IP (utilisée principalement pour la géolocalisation approximative).

2. Utilisation d'AdMob (Publicités)

Cette application utilise le service Google AdMob pour afficher des annonces publicitaires. Ces publicités nous permettent de proposer l'application gratuitement.

  • AdMob peut utiliser ces identifiants pour personnaliser les publicités en fonction de vos centres d'intérêt.
  • Pour en savoir plus sur la gestion de vos données par Google, vous pouvez consulter la Politique de confidentialité de Google.

3. Stockage local des statistiques

L'application enregistre vos statistiques de jeu (scores, progression) uniquement en local sur votre appareil.

  • Ces données de jeu ne sont pas partagées avec des tiers.
  • Elles sont supprimées automatiquement si vous désinstallez l'application.

4. Vos droits et choix

Vous avez le contrôle sur la personnalisation des publicités :

  • Sur Android : Allez dans Paramètres > Google > Annonces pour réinitialiser votre identifiant ou désactiver la personnalisation.
  • Sur iOS : Allez dans Réglages > Confidentialité > Publicité Apple.

5. Sécurité

Nous prenons les mesures nécessaires pour protéger les informations traitées localement. Les données publicitaires sont sécurisées par les protocoles de Google AdMob.
